# vins_ws

Public research workspace for ROS 2 VINS-related experiments.

## Repository layout

- `src/VINS-Fusion-ROS2-jazzy`: modified research fork of VINS-Fusion for ROS 2 Jazzy / Ubuntu 24.04
- `src/kaist_player`: auxiliary package used in this workspace
